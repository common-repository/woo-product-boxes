== Description ==

Product Boxes is an extension of WooCommerce, that allows to select products, and create a personalized bundles. 
Go to your admin area, create your product with Woocommerce, than go to "Included products" tab and select products.
In the site, in the product details, you can find a new tab called "Included products", with the product list.


== Installation ==

= Automatic installation =

Log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.
In the search field type Gift baskets and click Search Plugins. You can install it by simply clicking “Install Now”.

= Manual installation =

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Create a product.
4. In the "Included products" tab select the products you want to bundle together.
